Audit Automation

Pipeline d'automatisation des feuilles maîtresses d'audit à partir d'un FEC.

À propos

Cette application transforme un Fichier des Écritures Comptables (FEC) en feuilles maîtresses, balance comparative N / N-1 et feuilles de travail par cycle d'audit — automatiquement, pour n'importe quel client disposant d'un FEC français.

Lancer l'application

L'interface est une application Streamlit (serveur interactif). Elle se lance ainsi :

pip install -r requirements.txt
python3 -m streamlit run app.py

Ou via la ligne de commande :

python3 main.py FEC.txt --client NOM --date-cloture 31/12/2025 \
  --n1-fm FM_N-1.xlsx --templates data/templates/ --output output/

Hébergement

Streamlit nécessite un serveur applicatif persistant et ne peut pas s'exécuter en fonction serverless Vercel. Cette page statique sert de page de présentation. Pour héberger l'application interactive, utilisez une plateforme adaptée aux serveurs longue durée (Streamlit Community Cloud, Render, Railway, Hugging Face Spaces ou un serveur interne via Docker).
